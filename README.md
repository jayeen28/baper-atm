# Baper ATM

## Steps:
- Prompt for pin.
- If pin is correct.
- Display options
  - Balance enquiry
  - Cash withdraw.
  - Deposit cash.
  - Exit.

### Balance enquiry :
- Display with cash balance.

### Cash withdraw :
- Prompt for amount.
- Check the available balance.
- Debit the amount and show the available balance.

### Deposit cash :
- Prompt for amount.
- Show the avaiable balance and available + deposit amount.

### Exit
